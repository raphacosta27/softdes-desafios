    getInfo(user)

    Retorna as informações do usuário de acordo com o seu nome. Essa função é usada na rota /pass.

    Parameters:

    Name Type Description Default

    nome do usuário.

    		 required

    Returns:

    Type Description
    list

    lista: lista contendo as informações do usuário como senha e tipo

    getQuiz(id, user)

    Retorna o quiz de acordo com o número do id. Caso o usuário seja um administrador, é retornado todos os quizes. Caso contrário, são retornados apenas os quizes que estão em aberto. Essa função é usada na rota / (rota principal).

    Parameters:

    Name Type Description Default
    id int

    id do quiz.

    		 required
    user str

    nome do usuário.

    		 required

    Returns:

    Type Description
    list

    list: uma lista dos quizes.

    getQuizes(user)

    Retorna os quizes no banco de dados. Caso o usuário seja um administrador, é retornado todos os quizes. Caso contrário, são retornados apenas os quizes que estão em aberto. Essa função é usada na rota / (rota principal).

    Parameters:

    Name Type Description Default
    user str

    nome do usuário.

    		 required

    Returns:

    Type Description
    list

    list: uma lista de quizes com suas respectivas informações.

    getUserQuiz(userid, quizid)

    		 required
    quizid int

    id do quiz.

    		 required

    Returns:

    Type Description

    list: uma lista dos envios do usuário para o quiz específico

    lambda_handler(event, context)

    Executa o código enviado pelo aluno e testa de acordo com as repostas esperadas

    Parameters:

    Name Type Description Default
    event dict

    dicionário que possui as informações como o código do aluno, as respostas esperadas, os argumentos da funcao e o feedback.

    		 required
    context None

    não esta sendo usado

    		 required

    Returns:

    Type Description
    str

    uma string com os testes executados e seus resultados.
